2. thymeleaf

		웹 및 독립 실행형 환경 모두를 위한 최신 서버 즉 java 템플릿 엔진이다.
		▶ 웹 실행 == 요청 시 forward 되는 화면
		▶ 독립 실행 == html 파일 실행(인터넷 연결 x)
		
		HTML 파일에서 th(ThymeLeaf) 속성을 이용하여,
		컨트롤러로부터 전달받은 데이터를 사용해 동적 페이지를 만들 수 있다.
		
		Spring Boot 에서는 jsp 가 아닌 ThymeLeaf 사용을 권장하고 있다.
		
		[사용법]
			html 파일에서
				< !Doctype html > 밑에
				< html xmlns:th="https://www.thymeleaf.org" > 형식으로 작성하면 사용 가능하다.
				
				< html xmlns:별칭="템플릿을 가져올 주소" >
